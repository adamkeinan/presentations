Schema Customization

We can now create custom types and resolvers for your Gatsby sites.

Custom types (gatsby-node.js)

exports.sourceNodes = ({ actions }) => {
  actions.createTypes(`
    type MyCustomNode implements Node {
      name: String!
    }
  `);
};

Custom resolvers

exports.createResolvers = ({ createResolvers }) => {
  createResolvers({
    Query: {
      getGoodStuff: {
        resolve: () => {
          return ['whiskey', 'puppies', 'dessert']
        }
      }
    }
  })
};

Asset Prefixes

We can serve assets from a different CDN.

1. Update gatsby-config.js

module.exports = {
  assetPrefix: `https://cdn.example.com`,
}

2. Add the flag during builds

gatsby build --prefix-paths

3. There is no step 3

<script src="https://cdn.example.com/common-123.js"></script>
